1. Чем var/let/const в чем отличие, как передаются данные по параметрам, 
что такое передача по ссылке?
var - глобальная переменная, которая всплывает в документе. К ней можно обращаться в любой части док-та.
let и const - блочные переменные, они видны(можно ими пользоваться) только в том блоке, где их определили.
const - нельзя перезаписывать.
Данные передаются по значению - приметивные переменные, и по ссылке - объекты. Т.е. в последнем случае, 
мы передаем не сам объект,  а ссылку на него.
Что такое итератор, написать пример итератора?
Итератор - это такой объект, который имеет в свойствах метод next(). Благодаря которому, можно объект сделать 
перебираемым. Чтобы каждый раз не вызывать метод next(), надо добавить [Symbol.iterator] в объект. Он будет  
каждый раз вызывать метод next() и возвращать результат его работы, пока не дойдет до условия {done: true} .

const objIterator = {
first: 1,
last: 5,
[Symbol.iterator]() {
  return this;
},
next() {
  if(this.first <= this.last) {

    return {done: false, value: this.first++}
  }else 
    return{done: true, value: this.last}
}
};
